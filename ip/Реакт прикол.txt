Необходимо разработать прототип страницы страницы --------
Основные требования:
- Фон сайта - светло-серый.
- Форма -------- должна находится в карточке с белым фоном.
- Карточка должна располагаться по центру страницы при любом размере экрана или любом увеличении браузера.
- У карточки определено скругление, граница, -------- --------
- --------
- При нажатии на кнопку «--------» должна быть проверка на введённые данные. Если -------- состоят из 6 символов и больше - должно быть выведено сообщение об успехе, иначе - об ошибке.
Дополнительные требования:
- --------
- --------
- --------
- Карточка находится по центру страницы как по горизонтали, так и по вертикали.
- На странице нельзя использовать сторонние библиотеки для визуализации.
- Для позиционирования элементов используется flexbox.
- На странице определено изображение favicon.
- Для получения данных, введённых пользователем, используется класс FormData.
- Для отображения сообщений об успехе или ошибке входа используется библиотека Sweet Alert.
